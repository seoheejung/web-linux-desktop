COMPOSE DESIGN

단일 Compose 기반 실행 구성

image: lscr.io/linuxserver/webtop:ubuntu-kde container_name: web-linux-desktop environment: TZ: Asia/Seoul LC_ALL: ko_KR.UTF-8 CUSTOM_USER: ${WEBTOP_USER} PASSWORD: ${WEBTOP_PASSWORD} FILE_MANAGER_PATH: /workspace volumes: - webtop_config:/config - ./workspace:/workspace ports: - "127.0.0.1:3001:3001"

  • HTTPS 3001만 노출
